# ** Pokédex Project / Proyecto Pokédex**

***

## ** Description / Descripción**

**EN:**  
Java application that simulates a Pokédex for Pokémon trainers. It features an interactive menu to add, delete, search, and display Pokémon, plus advanced options like **Hack Pokédex** (convert all Pokémon to Ditto) and full database reset. Includes error handling for duplicates and invalid positions.

**ES:**  
Aplicación en Java que simula una Pokédex para entrenadores Pokémon. Incluye un menú interactivo para añadir, eliminar, buscar y mostrar Pokémon, además de opciones avanzadas como **Hack Pokédex** (convertir todos los Pokémon en Ditto) y reinicio completo de la base de datos. Implementa control de errores para evitar duplicados y posiciones inválidas.

***

## ** Key Features / Características principales**

*   ✅ **Add Pokémon at specific positions** / Añadir Pokémon en posiciones específicas
*   ✅ **Search and display all Pokémon** / Buscar y mostrar todos los Pokémon
*   ✅ **Pokémon counter & erase all data** / Contador y opción para borrar toda la información
*   ✅ **Auto-fill with test data** / Función para llenar automáticamente la Pokédex

***

## ** Installation / Instalación**

```bash
# Clone the repository / Clonar el repositorio
git clone https://github.com/youruser/pokedex.git

# Navigate to the project folder / Ir a la carpeta del proyecto
cd pokedex

# Compile the Java files / Compilar los archivos Java
javac Main.java

# Run the application / Ejecutar la aplicación
java Main
```
