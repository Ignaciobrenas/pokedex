# pokedex
Descripción:
Aplicación en Java que simula una Pokedex para entrenadores Pokémon. Incluye un menú interactivo con opciones para añadir, eliminar, buscar y mostrar Pokémon, así como funciones avanzadas como “Hack Pokedex” (convertir todos los Pokémon en Ditto) y reinicio completo de la base de datos. Se implementa control de errores para evitar duplicados y posiciones inválidas.
Características principales:

Añadir Pokémon en posiciones específicas.
Buscar y mostrar todos los Pokémon registrados.
Contador de Pokémon y opción para borrar toda la información.
Función especial para llenar automáticamente la Pokedex con datos de prueba.
