# pokedex
Aplicación en Java que simula una Pokedex para entrenadores Pokémon. Permite añadir, eliminar, buscar y mostrar Pokémon, además de funciones avanzadas como llenar automáticamente, contar registros, reiniciar datos y “Hack Pokedex” para convertir todos en Ditto. Incluye control de errores y menú interactivo.
